# Music Store Inventory

A full-stack inventory management application built with **Node.js**, **Express**, **PostgreSQL**, and **EJS** as part of The Odin Project.

## Live Demo

https://odin-inventory-application-ten.vercel.app/

## Features

- View all categories
- View items within a category
- Create, update, and delete categories
- Create, update, and delete inventory items
- PostgreSQL database integration
- Server-side rendering with EJS
- Deployed using Vercel and Neon

## Technologies Used

- Node.js
- Express.js
- PostgreSQL
- EJS
- CSS
- Neon Database
- Vercel

## Database Structure

The application uses three main tables:

- **Categories**
- **Items**
- **Brands**

Relationships are managed using PostgreSQL foreign keys and constraints.
